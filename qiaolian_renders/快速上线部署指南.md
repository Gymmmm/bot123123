# 快速上线部署指南（侨联版）

## 0) 前置条件

- 已有可用 UserBot Token（`.env` 中 `USER_BOT_TOKEN`）
- 已设置 `USER_BOT_USERNAME`（用于展示入口链接）
- 机器人可正常启动（`qiaolian-user-bot.service` active）

## 1) 发布频道置顶文案

1. 打开 `qiaolian_renders/频道置顶SEO推广文案.md`
2. 复制推荐版本发到频道并置顶
3. 将 `https://t.me/<你的UserBot用户名>?start=ch__channel_top` 替换为真实链接

## 2) 发布 Bot 固定索引

### 自动方式（推荐）

```bash
cd /opt/qiaolian_dual_bots
python3 qiaolian_renders/deploy_index.py
```

默认读取：
- `USER_BOT_TOKEN`
- `INDEX_CHAT_ID`（可选，不填时用 `ADMIN_CHAT_ID`）
- `ADMIN_CHAT_ID`（可选）

### 手动方式

1. 打开 `qiaolian_renders/Bot固定索引消息.md`
2. 复制主索引消息到 Bot 对话
3. 手工 Pin 该消息

## 3) 回归验收（最小集）

- `/start`：显示首页主导航
- `/find`：可进入找房流程
- `/appointments`：可查看预约页
- `/contact`：可进入顾问承接页
- 深链承接：
  - `a__<post_token>__l_<id>|cv=<a|b|c>`
  - `q__<post_token>__l_<id>|cv=<a|b|c>`
  - `ch__<topic>`
  - `t_bind_<code>`

## 4) 服务器侧检查命令

```bash
cd /opt/qiaolian_dual_bots
bash scripts/smoke_server.sh --host uhome-root --user root
```

若在服务器本机执行且无需 SSH，可只看 service：

```bash
systemctl is-active qiaolian-user-bot.service qiaolian-publisher-bot.service qiaolian-collector.service qiaolian-admin-web.service
```

## 5) 常见问题

- **409 Conflict**
  - 同一 bot token 被多个进程同时轮询；保留 systemd 单实例，关闭其他实例。

- **频道链接进 Bot 后无上下文**
  - 检查发布端按钮深链是否为 `a__/q__` 标准格式。

- **索引消息无法固定**
  - 需要在目标会话具备固定权限（频道/群中需管理员权限）。
