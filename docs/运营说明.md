## 1. 频道整体结构（先看这里）

把频道想象成一本书：**置顶 = 封面/目录**，**汇总帖 = 各章节**，**每条新房源 = 正文页**。

```
频道
├── 📌 置顶帖（总目录）        ← 全频道唯一置顶，用户第一眼就看到
│     ├── 链接→ BKK1 汇总帖
│     ├── 链接→ BKK2 汇总帖
│     ├── 链接→ 钻石岛/市中心 汇总帖
│     ├── 链接→ 别墅/独栋 汇总帖
│     └── 链接→ 已租/下架 汇总帖
│
├── 📋 BKK1 汇总帖             ← 长期不动，只在底部追加新行
├── 📋 BKK2 汇总帖
├── 📋 钻石岛/市中心 汇总帖
├── 📋 别墅/独栋 汇总帖
├── 📋 已租/下架 汇总帖
│
├── 🏠 新房源帖（每天新增）    ← 每套房源独立一条消息（有图有价有联系方式）
├── 🏠 新房源帖
└── ...
```

**核心原则（只有 3 条）：**
1. **置顶只放 1 条总目录帖**，不要置顶多条消息——否则置顶区很乱。
2. **每个区域/分类保留 1 条"汇总帖"**，长期追加，不要每次新建。
3. **每套房源单独发 1 条消息**，带图带价格，已租/下架在原帖里标注。

## 2. 置顶怎么做

### 2.1 如何置顶一条消息

**手机操作：**
1. 进入你的频道。
2. 找到你要置顶的那条消息（总目录帖）。
3. **长按**那条消息，弹出菜单。
4. 点选 **"置顶（Pin）"**。
5. 选 **"对所有成员置顶"**，点确认。

**电脑/网页操作：**
1. 进入频道，找到目标消息。
2. 鼠标**右键**点击那条消息。
3. 点 **"置顶（Pin Message）"**。
4. 勾选 **"通知所有成员"**（可选，建议勾），点确认。

> **注意：** 如果之前已经有多条置顶，先把多余的解除置顶（长按 → 取消置顶），只保留**总目录帖**这 1 条。

## 3. 目录/索引帖怎么建

### 3.1 第一步：先建好各「汇总帖」

每个分类（如 BKK1、BKK2……）先单独发一条空帖，作为该分类的"汇总帖"。

> **示例——BKK1 汇总帖内容：**
> ```
> 📋【BKK1 房源汇总】
> ┌─────────────────────────────────
> 日期       户型   月租    状态   跳转
> ─────────────────────────────────
> 2026-04-25 两房   $1,200  🟢新   （↗链接）
> 2026-04-20 一房   $650    🟢新   （↗链接）
> 2026-04-15 三房   $1,800  🔴已租
> ─────────────────────────────────
> 更新：每有新房追加一行；已租/下架标🔴
> ```

### 3.2 第二步：拿到汇总帖的「消息链接」

1. 发好汇总帖之后，**长按**那条消息（电脑右键）。
2. 点 **"复制链接（Copy Link）"**。
3. 得到类似这样的链接：`https://t.me/你的频道名/消息编号`

### 3.3 第三步：建「总目录帖」并插入链接

发一条新消息，把所有汇总帖的链接写进来：

> **示例——总目录帖内容：**
> ```
> 📌【频道导航】金边租房频道
> ─────────────────────────
> 点下方链接直接跳到对应分类：
>
> 🏙 BKK1 房源汇总 → https://t.me/xxx/101
> 🏙 BKK2 房源汇总 → https://t.me/xxx/102
> 🌴 钻石岛/市中心 汇总 → https://t.me/xxx/103
> 🏠 别墅/独栋 汇总 → https://t.me/xxx/104
> 🔴 已租/下架 汇总 → https://t.me/xxx/105
>
> ─────────────────────────
> 💡 使用方法
>   · 点上面链接 → 进入对应分类汇总
>   · 汇总帖里每行有 ↗链接 → 直接看房源详情
>   · 有意向：私信 @管理员 或按房源帖下方联系方式
>
> ⚠️ 发布需求/房源：请按模板私信管理员 @xxx
> 最后更新：2026-04-25
> ```

### 3.4 第四步：把总目录帖置顶（参考第 2 节）

### 3.5 如何更新目录帖中的链接

每当增加了新分类汇总帖时：
1. 发新汇总帖 → 复制链接。
2. 打开总目录帖 → **长按 → 编辑（Edit）** → 追加一行新链接 → 保存。

> **注意：** Telegram 频道管理员才能编辑已发消息；普通成员不能编辑。

## 4. 新房源怎么发

### 4.1 发布格式（统一模板，直接复制）

```
🏠【新房源｜BKK1｜两房｜$1,200/月】

📍 地址/地标：BKK1，靠近 XXX 商场
🏠 房型/面积：两房两厅 / 约 65㎡
🛋 家具家电：全配（床/沙发/冰箱/洗衣机/空调）
💰 月租：$1,200（可小刀）
📑 押金：押二付一
📅 合同期：6 个月起
✅ 可入住：2026-05-01
⚡ 电费：按表（约 $0.25/度）
🚰 水费：按表

✨ 亮点：楼下超市 / 带泳池 / 宠物友好

📞 联系方式：@管理员 / +855 xx xxxx xxxx
```

> **提示：** 发布时同步附上 3～5 张实拍照片，无图的房源很难引起兴趣。

### 4.2 发布后需要做的 2 件事

1. **复制这条房源帖的链接**（长按 → 复制链接）。
2. **打开对应区域的汇总帖** → **长按 → 编辑** → 在列表底部追加一行：

   ```
   2026-04-25  两房  $1,200  🟢新   https://t.me/xxx/999
   ```

这样用户从目录进汇总帖，就能看到最新房源的跳转链接。

## 5. 历史房源怎么处理

你有几百条历史房源，不需要一条条去看——用下面**分批整理法**：

### 5.1 方案（推荐）：建「历史房源总索引帖」

1. 发一条「历史房源汇总」帖：

   ```
   📦【历史房源汇总（2023～2025）】
   以下为频道建立以来的旧房源链接，仅供参考。
   绝大多数已租出，如需了解相似房型请联系 @管理员。

   ── 2025 年 ──
   2025-12   BKK1 两房 $1,100   https://t.me/xxx/88
   2025-11   BKK2 一房  $600    https://t.me/xxx/77
   ...（按月份倒序追加）

   ── 2024 年 ──
   ...

   ── 2023 年 ──
   ...
   ```

2. 把这条帖的链接加进**总目录帖**里（"历史房源" 一行）。

### 5.2 不需要手动整理全部历史的情况

如果几百条历史帖整理工作量太大，可以：
- 只在**总目录帖**里写一句：
  > 📦 历史房源（2023～2025）：频道内搜索 `#旧房源` 或直接上下滑动查阅。
- 对 **今后新发的房源** 严格按模板来，保证新内容整洁即可。

> 历史帖不需要删除，Telegram 频道消息不会影响新帖展示，只要目录指引清晰就行。

## 6. 已租/下架房源怎么标注

### 6.1 原帖标注（最重要的一步）

当一套房已租出或下架时：
1. 找到该房源的帖子。
2. **长按 → 编辑**。
3. 在标题第一行加上状态：

   | 状态   | 标注示例                                              |
   |--------|-------------------------------------------------------|
   | 已租出 | `🔴【已租｜BKK1｜两房｜$1,200/月】`                  |
   | 已下架 | `⛔【下架｜BKK1｜两房｜$1,200/月】`                  |
   | 价格调整 | `🟡【调价→$1,100｜BKK1｜两房】`                    |

4. 在正文底部追加一行：
   ```
   ─────────────
   状态更新：2026-04-28 已租出
   ```

### 6.2 更新汇总帖中的对应行

打开该区域的汇总帖 → 编辑 → 把对应行的「🟢新」改成「🔴已租」或「⛔下架」：

```
2026-04-25  两房  $1,200  🔴已租  https://t.me/xxx/999
```

### 6.3 「已租/下架汇总帖」（可选，便于统计）

你也可以维护一条专门的「已租/下架汇总帖」，把已成交房源统一整理在这里，便于做业绩统计或向房主反馈。

## 7. 每日/每周维护流程

### 每日操作（约 5 分钟）

```
新房源：
  1. 按模板发新帖（含图片）
  2. 复制该帖链接
  3. 打开对应汇总帖 → 编辑 → 追加一行

有房已租/下架：
  4. 找到原帖 → 编辑标题加🔴/⛔
  5. 更新对应汇总帖里的状态
```

### 每周操作（约 10 分钟）

```
  6. 检查汇总帖，确认所有已租/下架都已更新状态
  7. 检查总目录帖，如有新分类则添加链接
  8. 更新总目录帖底部的"最后更新"日期
```

### 每月操作（约 20 分钟）

```
  9. 把「本月已租/下架」的行，从各汇总帖移到「历史房源汇总帖」
      （或只更新状态，不移动也可以——看你风格）
 10. 检查有无价格过期、联系方式失效的房源
```

## 8. 标签系统说明

### 是否需要标签（#tag）？

| 情况 | 建议 |
|------|------|
| 你愿意每条都手动加标签 | 可以用，有利于频道内搜索 |
| 你只想靠"汇总帖+链接"维护 | 不用强制加，汇总帖足够 |
| 两种都有最好 | 推荐：发帖时加标签，同时更新汇总帖 |

### 推荐标签体系（如果用的话，固定这套，不要随意变化）

```
按区域：  #BKK1  #BKK2  #钻石岛  #市中心  #金边
按户型：  #一房  #两房  #三房以上  #别墅  #合租  #店铺
按预算：  #500以下  #500_1000  #1000_1500  #1500以上
按状态：  #新房源  #已租  #下架  #旧房源
其他：    #视频看房  #可短租  #宠物友好
```

**使用说明：**
- 每条房源帖的**第一行**放标签，例如：
  ```
  #BKK1 #两房 #1000_1500 #新房源
  ```
- 用户在频道右上角 🔍 搜索 `#BKK1` 或 `#两房`，就能筛出同类房源。
- **标签写法要统一**：一直用 `#1000_1500`，不要一会儿 `#1000-1500`，一会儿 `#1000到1500`。

## 9. 运营建议总结

| 问题 | 解决方案 |
|------|----------|
| 用户找不到房源 | 总目录帖置顶 + 各分类汇总帖 + 教用户点链接/搜索 |
| 历史房源太多太乱 | 建「历史房源汇总帖」，新房源严格按模板，历史帖不用删 |
| 已租房源没人知道 | 发现已租立即编辑原帖标题加🔴，同步更新汇总帖 |
| 频道越来越乱 | 只置顶 1 条总目录帖；发帖统一用同一模板 |
| 房主/租客找到联系方式难 | 每条房源帖底部必须有联系方式；总目录帖也要有 |
| 有人在频道乱发消息 | 频道设为"只有管理员可发言"；开一个关联群给用户留言 |

### 给用户的「使用说明」（可直接放在总目录帖里）

```
💡 怎么找房：
   1. 点上方目录里对应区域链接
   2. 汇总帖里找到你想看的房源，点 ↗链接 看详情
   3. 在频道右上角 🔍 搜索：如 #BKK1 / #两房 / $1000

📞 怎么联系：
   · 看中哪套房，直接在该帖下方评论，或联系 @管理员
   · 有房要出租：私信 @管理员，按模板提交

⚠️ 注意：
   · 频道内只发房源公告，互动请去关联讨论群
   · 虚假房源/诈骗即封
```
