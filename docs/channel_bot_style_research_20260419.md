# 同类型频道/机器人样式调研（2026-04-19）

## 样本

1. `RentalKhChannel`（金边租房频道）
- 观察：帖子普遍是「区域/户型/价格 + 简短亮点 + 联系方式」的短结构，易扫读。
- 可借鉴：首屏就给价格和地点，避免长段解释。

2. `slovenia_real_estate_en` + 配套 bot（`@slovenia_real_estate_bot`）
- 观察：频道里明确引导用户用 bot 做「个性化通知」。
- 可借鉴：频道负责展示，bot 负责承接和筛选，入口要短、动作要明确。

3. `MLS Dubai Real Estate`（配套 bot：`@MLS_Dubai_Real_Estate_bot`）
- 观察：频道品牌感统一，bot 作为直接咨询入口。
- 可借鉴：频道视觉统一 + 单一主 CTA，降低选择疲劳。

## 本仓落地策略

1. 文化表达（简短、可信）
- 统一使用「实拍优先、费用先说、可逐项确认」三类词，减少空泛营销词。
- 新用户不灌输长文，默认给 3 步动作：找房 → 预约 → 咨询。

2. 视觉表达（可扫读）
- 频道帖子维持 8 行结构，不加长段落。
- 主 bot 首页增加「快速开始」按钮，减少用户第一次操作成本。
- 无匹配结果时提供明确下一步按钮，避免对话中断。
